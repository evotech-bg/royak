Deployed by a git push, built and served by Royak's own ingress. Every number below is this cluster's real, live state — polled from Royak's API every 2 seconds.

This response served by pod __POD__ · now — refresh and Royak's ingress round-robins you to a different replica. Not a static page.

A chaos monkey is loose on purpose. Royak kills a random pod every ~20 seconds so you can watch it notice and self-heal — every time, automatically. Nothing here is broken; the pods dying and coming back is the demo. Every number below is this live cluster.
⚡
In front of Royak sits NeuroCaddy — our own Rust web server + TLS edge. It terminates HTTPS (Let's Encrypt) and load-balances across replicas; a small model watches this site's traffic and flags statistical anomalies (z-score outlier detection). Edge, orchestrator and cross-node mesh — the whole stack is homegrown.
